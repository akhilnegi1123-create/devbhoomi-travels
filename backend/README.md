# 🏔️ DevBhoomi Travels — Backend

## Chalane ke liye sirf 3 commands:

```
cd backend
npm install
npm start
```

## Kya kya kaam karta hai:
- ✅ Signup / Login (JWT)
- ✅ Booking submit
- ✅ Contact form
- ✅ Reviews (submit + approve)
- ✅ Newsletter subscribe
- ✅ Admin panel ka data
- ✅ Frontend serve karta hai

## Important URLs:
- Website     →  http://localhost:5000
- Admin Panel →  http://localhost:5000/admin.html
- API Health  →  http://localhost:5000/api/health
